# IoT Based Battery Management System

## Components Used
- ESP32
- OLED SSD1306
- DS18B20 Temperature Sensor
- Relay Module
- Buzzer
- LEDs
- Potentiometer
- ThingSpeak Cloud

## Features
- Battery Voltage Monitoring
- Battery Percentage Calculation
- Temperature Monitoring
- Relay Protection
- Buzzer Alert
- OLED Display
- ThingSpeak IoT Monitoring

## Software Used
- Arduino IDE
- Wokwi Simulator
- ThingSpeak
